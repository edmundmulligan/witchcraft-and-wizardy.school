# How to Create a Projects Folder on Windows

## What is a Projects Folder?

A Projects folder is a special place on your computer where you keep all your coding projects organised. It's like having a toy box where you keep all your toys in one place - it makes everything easier to find!

## Why Do You Need It?

When you start coding, you'll create lots of different projects. Having them all in one folder helps you:
- ✅ Find your projects quickly
- ✅ Keep your computer organised
- ✅ Back up your work easily
- ✅ Know where to save new projects

## Method 1: Using File Explorer (Easiest!)

This is the easiest way to create your Projects folder.

### Step 1: Open File Explorer

1. Click the **folder icon** on your taskbar (bottom of screen)
   - Or press the **Windows key + E** on your keyboard

### Step 2: Go to Your User Folder

1. Look on the left side of File Explorer
2. Click on **"This PC"** or **"Quick access"**
3. Find and click on your **user folder** (it has your name and a person icon)
   - It might be called something like "Documents" or your Windows username

### Step 3: Create the Projects Folder

1. Right-click in an empty space in the window
2. Hover over **"New"** in the menu that appears
3. Click **"Folder"**
4. A new folder appears with the name highlighted
5. Type: **Projects**
6. Press **Enter**

🎉 **Done!** You now have a Projects folder!

## Method 2: Using Command Prompt

This method uses commands, which is more like "real programming"!

### Step 1: Open Command Prompt

1. Click the **Start** button
2. Type **"cmd"** or **"Command Prompt"**
3. Click to open Command Prompt (a black window will appear)

### Step 2: Create the Projects Folder

1. Type this command and press **Enter**:
   ```
   mkdir Projects
   ```
   - `mkdir` stands for "make directory" (directory is another word for folder)

2. To check it was created, type:
   ```
   dir
   ```
   - You should see "Projects" in the list

3. To open it, type:
   ```
   cd Projects
   ```
   - `cd` stands for "change directory" (move into that folder)

🎉 **Done!** Your Projects folder is created!

## Where is Your Projects Folder?

Your Projects folder is in your **user folder**, which is usually here:
```
C:\Users\YourName\Projects
```
(Replace "YourName" with your actual Windows username)

## How to Find It Again

### Using File Explorer:
1. Open File Explorer
2. Click on your user folder on the left
3. Look for the **Projects** folder

### Using Quick Access (Even Faster!):
1. Open File Explorer
2. Find your Projects folder
3. **Right-click** on it
4. Choose **"Pin to Quick access"**
5. Now it will always be at the top of the left side!

## Creating Folders Inside Projects

As you start coding, you'll want to create a new folder for each project:

1. Open your **Projects** folder
2. Right-click in empty space
3. Choose **New** → **Folder**
4. Give it a name like:
   - `my-first-website`
   - `javascript-practice`
   - `game-project`
   - `school-assignment`

**Tip:** Use lowercase letters and dashes (-) instead of spaces in project names. This makes them easier to work with in code!

## Organising Your Projects

Here's how your folder structure might look:

```
Projects/
├── my-first-website/
├── javascript-practice/
├── python-games/
└── school-projects/
```

Each project gets its own folder. Nice and organized!

## Using Your Projects Folder

### When starting a new project:
1. Open your Projects folder
2. Create a new folder for the project
3. Open that folder
4. Start creating your files!

### From Command Prompt:
```
cd Projects
mkdir my-new-project
cd my-new-project
```

### From VS Code:
1. Open VS Code
2. Click **File** → **Open Folder**
3. Navigate to **Projects**
4. Choose the project folder you want to work on

## Tips for Staying Organized

- 📁 Create a new folder for each project
- 📝 Give folders clear names so you remember what they are
- 🗑️ Delete old practice projects you don't need anymore
- 💾 Back up your Projects folder regularly (copy it to a USB drive or cloud storage)
- 📅 Add dates to folder names if helpful: `2026-05-school-project`

## Common Problems and Solutions

### "I can't find my Projects folder!"
- Open File Explorer
- Click in the search box (top right)
- Type "Projects"
- Windows will find it for you

### "I made it in the wrong place!"
- You can drag and drop the folder to move it
- Or create a new one in the right place and delete the old one

### "I want to delete it and start over"
- Right-click the Projects folder
- Choose "Delete"
- Follow the steps above to create a new one

## What's Next?

Now that you have your Projects folder:
1. ✅ Install VS Code (if you haven't already)
2. ✅ Install Node.js (if you need it)
3. ✅ Create your first project folder inside Projects
4. ✅ Start coding!

## Need Help?

If something doesn't work:
1. Make sure you typed the folder name correctly
2. Try using File Explorer if Command Prompt didn't work (or vice versa)
3. Ask an adult or teacher for help
4. Remember: You can always delete the folder and try again!

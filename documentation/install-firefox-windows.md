# How to Install Firefox on Windows

## What is Firefox?

Firefox is a free web browser made by Mozilla. A web browser is the program you use to visit websites on the internet - like Google Chrome, Microsoft Edge, or Safari.

## Before You Start: Check if Firefox is Already Installed

Before downloading, let's see if you already have Firefox on your computer!

### Method 1: Check the Start Menu

1. Click the **Start** button (Windows logo in the bottom-left corner)
2. Type **"Firefox"**
3. If you see it appear in the search results with an orange/red fox icon, **it's already installed!** 
   - You can click it to open Firefox
   - **Skip to "What's Next?" at the bottom of this guide**
4. If you don't see it, continue to the installation steps below

### Method 2: Check Your Desktop or Taskbar

1. Look at your **desktop** (the main screen) - do you see a Firefox icon? (orange/red fox wrapped around a blue globe)
2. Look at your **taskbar** (the bar at the bottom of your screen) - is Firefox pinned there?
3. If you find it, **Firefox is already installed!** You can skip the installation.

## Installation Steps

**Only follow these steps if Firefox is NOT already installed on your computer.**

### Step 1: Download Firefox

1. Open your current web browser (like Edge or Chrome)
2. Go to: **https://www.mozilla.org/firefox**
3. Click the big **"Download Firefox"** button
4. Wait for the file to download (it's called something like `Firefox Installer.exe`)

### Step 2: Run the Installer

1. Find the downloaded file in your **Downloads** folder
2. **Double-click** the file to start the installation
3. If Windows asks "Do you want to allow this app to make changes?", click **Yes**

### Step 3: Follow the Setup Wizard

1. Click **Install** on the welcome screen (the installer uses smart defaults)
2. Wait for the installation to finish (this takes about 30 seconds to 1 minute)
3. Firefox will open automatically when installation is complete
4. Congratulations! You've installed Firefox! 🎉

### Step 4: First Time Opening Firefox

1. Firefox will show a welcome page
2. You might be asked:
   - **"Make Firefox your default browser?"** - This means Firefox will open when you click links. You can choose **Yes** or **Skip** (you can change this later)
   - **"Set Up Sync?"** - This lets you save bookmarks to your account. You can choose **Not now** if you want (you can do this later)
3. That's it! Firefox is ready to use!

## What's Next?

Now that you have Firefox installed, you can:
- Start browsing the web
- Set your favorite website as your homepage
- Install add-ons (extra features) to customize Firefox
- Import bookmarks from your old browser

## Need Help?

If something doesn't work:
1. Make sure you downloaded Firefox from the official website (mozilla.org/firefox)
2. Try restarting your computer
3. Check if you have enough space on your computer (Firefox needs about 200MB)
4. Ask an adult or teacher for help

## Tips for Beginners

- **Zoom in/out:** Hold `Ctrl` and press `+` or `-`
- **New tab:** Press `Ctrl + T`
- **Close tab:** Press `Ctrl + W`
- **Bookmarks:** Press `Ctrl + D` to save a website you like
- **Private browsing:** Press `Ctrl + Shift + P` (this doesn't save your history)
- **Homepage:** You can set any website as your homepage in Settings

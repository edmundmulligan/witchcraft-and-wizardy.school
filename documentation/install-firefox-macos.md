# How to Install Firefox on macOS

## What is Firefox?

Firefox is a free web browser made by Mozilla. A web browser is the program you use to visit websites on the internet - like Safari, Google Chrome, or Microsoft Edge.

## Before You Start: Check if Firefox is Already Installed

Before downloading, let's see if you already have Firefox on your computer!

### Method 1: Check the Applications Folder

1. Open **Finder** (click the smiling face icon in your Dock)
2. Click **Applications** in the left sidebar
3. Scroll through the list looking for **Firefox** (orange/red fox icon)
4. If you find it, **Firefox is already installed!** 
   - You can double-click it to open Firefox
   - **Skip to "What's Next?" at the bottom of this guide**
5. If you don't see it, continue to the installation steps below

### Method 2: Check with Spotlight

1. Press **Command (⌘) + Space** to open Spotlight
2. Type **"Firefox"**
3. If you see it appear with an orange/red fox icon, **it's already installed!**
   - Press **Return** to open Firefox
   - **Skip to "What's Next?" at the bottom of this guide**
4. If you don't see it, continue to the installation steps below

## Installation Steps

**Only follow these steps if Firefox is NOT already installed on your computer.**

### Step 1: Download Firefox

1. Open your current web browser (like Safari)
2. Go to: **https://www.mozilla.org/firefox**
3. Click the big **"Download Firefox"** button
4. Wait for the file to download (it's called something like `Firefox xx.x.dmg`)
5. The file will be saved to your **Downloads** folder

### Step 2: Install Firefox

1. Find the **Firefox.dmg** file in your Downloads folder (or click it in your Downloads list)
2. **Double-click** the .dmg file to open it
3. A window will appear showing the **Firefox** icon and an **Applications** folder
4. **Drag the Firefox icon** into the **Applications** folder
5. Wait for Firefox to copy (this takes about 10-30 seconds)
6. Once copying is complete, **eject the Firefox disk image**:
   - Click the **eject button** next to "Firefox" in Finder's sidebar
   - Or drag the Firefox disk image to the Trash

### Step 3: Open Firefox for the First Time

1. Open **Finder**
2. Click **Applications**
3. Find and **double-click Firefox**
4. You might see a message: **"Firefox is an app downloaded from the internet. Are you sure you want to open it?"**
   - Click **Open** (this is safe - Firefox is from Mozilla's official website)
5. Firefox will open! 🎉

### Step 4: First Time Setup

1. Firefox will show a welcome page
2. You might be asked:
   - **"Make Firefox your default browser?"** - This means Firefox will open when you click links. You can choose **Set as Default** or **Not now** (you can change this later)
   - **"Set Up Firefox Sync?"** - This lets you save bookmarks to your account. You can choose **Not now** if you want (you can do this later)
3. That's it! Firefox is ready to use!

## Optional: Add Firefox to Your Dock

To make Firefox easier to find:

1. With Firefox open, look at its icon in the Dock
2. **Right-click** (or Control-click) the Firefox icon in the Dock
3. Choose **Options** → **Keep in Dock**
4. Now Firefox will always stay in your Dock!

## What's Next?

Now that you have Firefox installed, you can:
- Start browsing the web
- Set your favorite website as your homepage
- Install add-ons (extra features) to customize Firefox
- Import bookmarks from Safari or your old browser

## Need Help?

If something doesn't work:
1. Make sure you downloaded Firefox from the official website (mozilla.org/firefox)
2. Try restarting your computer
3. Check if you have enough space on your computer (Firefox needs about 200MB)
4. Make sure you're running a supported version of macOS
5. Ask an adult or teacher for help

## Tips for Beginners

- **Zoom in/out:** Hold `Command (⌘)` and press `+` or `-`
- **New tab:** Press `Command (⌘) + T`
- **Close tab:** Press `Command (⌘) + W`
- **Bookmarks:** Press `Command (⌘) + D` to save a website you like
- **Private browsing:** Press `Command (⌘) + Shift + P` (this doesn't save your history)
- **Homepage:** You can set any website as your homepage in Settings

## Keyboard Shortcuts for macOS

- **Command (⌘) + Q** - Quit Firefox completely
- **Command (⌘) + N** - Open a new window
- **Command (⌘) + ,** (comma) - Open Firefox Settings
- **Command (⌘) + L** - Select the address bar
- **Command (⌘) + [** or **]** - Go back or forward

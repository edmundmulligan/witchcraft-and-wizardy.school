# How to Install Visual Studio Code on Windows

## What is Visual Studio Code?

Visual Studio Code (VS Code) is a free program that helps you write code. Think of it like Microsoft Word, but for programming!

## Before You Start: Check if VS Code is Already Installed

Before downloading, let's see if you already have VS Code on your computer!

### Method 1: Check the Start Menu

1. Click the **Start** button (Windows logo in the bottom-left corner)
2. Type **"Visual Studio Code"** or **"VS Code"**
3. If you see it appear in the search results with a blue icon, **it's already installed!** 
   - You can click it to open VS Code
   - **Skip to "What's Next?" at the bottom of this guide**
4. If you don't see it, continue to the installation steps below

### Method 2: Check Using Command Prompt

1. Click the **Start** button
2. Type **"cmd"** or **"Command Prompt"**
3. Click to open Command Prompt (a black window will appear)
4. Type this command and press **Enter**:
   ```
   code --version
   ```
5. **If you see version numbers** (like 1.95.0), VS Code is already installed! Close this window and skip the installation.
6. **If you see an error** like "'code' is not recognised", VS Code is not installed yet - continue to the next section.

## Installation Steps

**Only follow these steps if VS Code is NOT already installed on your computer.**

### Step 1: Download VS Code

1. Open your web browser (like Chrome or Edge)
2. Go to: **https://code.visualstudio.com**
3. Click the big blue **"Download for Windows"** button
4. Wait for the file to download (it's called something like `VSCodeUserSetup-x64-1.xx.x.exe`)

### Step 2: Run the Installer

1. Find the downloaded file in your **Downloads** folder
2. **Double-click** the file to start the installation
3. If Windows asks "Do you want to allow this app to make changes?", click **Yes**

### Step 3: Follow the Setup Wizard

1. Click **Next** on the welcome screen
2. Click **I accept the agreement**, then click **Next**
3. Keep clicking **Next** (the default settings are fine)
4. On the "Select Additional Tasks" page, make sure these boxes are **checked**:
   - ✅ Create a desktop icon
   - ✅ Add "Open with Code" action to Windows Explorer file context menu
   - ✅ Add "Open with Code" action to Windows Explorer directory context menu
   - ✅ Add to PATH
5. Click **Next**, then click **Install**
6. Wait for the installation to finish (this takes about 1-2 minutes)
7. Click **Finish** (you can leave "Launch Visual Studio Code" checked)

### Step 4: First Time Opening VS Code

1. VS Code will open automatically (or double-click the desktop icon)
2. You'll see a welcome screen - you can close this for now
3. Congratulations! You've installed VS Code! 🎉

## What's Next?

Now that you have VS Code installed, you can:
- Open a folder to start coding
- Install extensions to add more features
- Start writing your first program!

## Need Help?

If something doesn't work:
1. Make sure you downloaded VS Code from the official website (code.visualstudio.com)
2. Try restarting your computer
3. Ask an adult or teacher for help

## Tips for Beginners

- **Save your work often!** Press `Ctrl + S` to save
- **Zoom in/out:** Hold `Ctrl` and scroll your mouse wheel
- **Open a folder:** Click `File` → `Open Folder` at the top
- **Dark mode:** VS Code starts in dark mode (black background). If you prefer light mode, click the gear icon ⚙️ at the bottom left → `Themes` → `Color Theme` → choose "Light+"

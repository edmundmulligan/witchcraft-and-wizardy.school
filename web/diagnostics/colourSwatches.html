Web Witchcraft and Wizardry Colour Swatches

Home > Diagnostics > Colour Swatches

This page displays all theme and style colours used in the Web Witchcraft and Wizardry project. The colour system includes three style variants (normal, subdued, and vibrant) with both light and dark themes. All colours are defined using HSL format and stored as CSS custom properties in styles/definitions/colours.css.

Base Colours

Base colours are not used directly in the themes but serve as the foundation for generating other colours. Colours at 90 and 150 degrees are not defined as they are too close in perceived appearance to green at 120 degrees to provide useful variation.

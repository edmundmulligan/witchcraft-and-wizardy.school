Web Witchcraft and Wizardry Colour Palette

Home > Diagnostics > Colour Palette

This page displays all colour variables defined in colours.css. The colour system includes three style variants (normal, subdued, and vibrant) with both light and dark themes. All colour combinations meet at least WCAG 2.2 AA contrast standards (4.5:1 ratio). Normal theme and popovers meet AAA standards (7:1 ratio). Each card shows the background colour with its corresponding text colour applied, along with both CSS variable names and sample text to preview readability.

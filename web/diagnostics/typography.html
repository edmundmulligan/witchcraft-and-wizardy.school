Responsive Font Sizes

All font sizes use clamp(min, preferred, max) for fluid responsive scaling.

--fontsize-large
clamp(1rem, 3vw + 1rem, 3rem)
Large Text Display
Used for prominent headings and hero text. Scales from 1rem (16px) to 3rem (48px).
--fontsize-normal
clamp(0.5rem, 2vw + 1rem, 2.25rem)
Normal Body Text
Standard body text size. Scales from 0.5rem (8px) to 2.25rem (36px).
--fontsize-small
clamp(0.5rem, 1.5vw + 0.75rem, 1.75rem)
Small Text Display
Used for captions and fine print. Scales from 0.5rem (8px) to 1.75rem (28px).
--fontsize-header-large
clamp(0.625rem, 1.5vw + 0.75rem, 2.5rem)
Large Header
Large section headers. Scales from 0.625rem (10px) to 2.5rem (40px).
--fontsize-header-medium
clamp(0.625rem, 1.5vw + 0.75rem, 2.5rem)
Medium Header
Medium section headers. Scales from 0.625rem (10px) to 2.5rem (40px).
--fontsize-header-small
clamp(0.5rem, 1.25vw + 0.625rem, 2rem)
Small Header
Small section headers. Scales from 0.5rem (8px) to 2rem (32px).
--fontsize-button
clamp(0.4rem, 1.75vw + 0.75rem, 2rem)
Button Text
Text inside buttons. Scales from 0.4rem (6.4px) to 2rem (32px).
--fontsize-nav
clamp(0.5rem, 1.5vw + 0.5rem, 1rem)
Navigation Links
Navigation menu text. Scales from 0.5rem (8px) to 1rem (16px).

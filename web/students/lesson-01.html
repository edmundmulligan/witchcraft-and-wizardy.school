Your First Web Page

HTML is the standard language used to create web pages. It consists of a series of elements that define the structure and content of a web page. Unlike a word processor, where you format text visually (that is, if you want bold text you simply highlight it and click a button to make it bold), in HTML you use tags to mark up the text and these tags tell the browser how to display the content. This is similar to how wizards and witches use spells to transform objects; in HTML, you use tags to transform plain text into structured web content.

Usually tags come in pairs, with an opening tag and a closing tag. The opening tag is written with angle brackets, like <tagname>, and the closing tag is similar but includes a forward slash before the tag name, like </tagname>. The content goes between the opening and closing tags. For example, to create a paragraph, you would use the <p> tag. Here is an example: <p>This is a paragraph.</p>.

What does it mean

In this section, we will explain what the different parts of the HTML code you wrote mean.

  • Line 1: <!DOCTYPE html>: This declaration defines the document type and version of HTML. It helps the browser understand how to interpret the content. Every HTML document should start with this declaration.
  • Line 2:<html lang="en">: This is the root element of the HTML document. The lang attribute specifies the language of the document, which is English in this case. Note that there is a </html> tag at the end of the document in line 14 to close this element.
  • Lines 3-7: <head> section: This section contains information about the document that is not displayed by the browser but helps the browser process the document correctly. This includes the character encoding, viewport settings for responsive design, and the title of the page that appears in the browser tab. For now you can treat this section as part of the incantation you need to get the page to display correctly. In later lessons you will add more to this section to load style sheets and javascript programs. The </head> tag closes this section.
  • Lines 8-11: <body> section: This section contains the actual content of the web page that is displayed in the browser. Everything you want to show on the page goes inside the <body> tags. The </body> tag closes this section.
  • Line 9: <h1>Hello, Witches and Wizards!</h1>: This line creates a heading. The <h1> tag defines a top-level heading, which is usually the largest and most important heading on the page. The text between the opening and closing tags is what will be displayed as the heading.
  • Line 10: <p>You are all awesome.</p>: This line creates a paragraph. The <p> tag defines a paragraph of text. Similar to the heading, the text between the opening and closing tags is what will be displayed as the paragraph content.

Be Curious

This is a chance to experiment and see what happens when you change things in your HTML code. Try modifying the code you wrote earlier in different ways, such as:

  • Change the text inside the <h1> and <p> tags to something else.
  • Add more paragraphs using the <p> tag.
  • Try adding other heading levels like <h2> or <h3> and see how they look.
  • Try putting in deliberate mistakes in your code and see what happens. Sometimes Visual Studio Code will detect the error and warn you about it. But sometimes the code will still run in the browser despite the mistakes.
  • Try adding comments in different places in your code. Comments start with 
    <!--
     and end with 
    -->
    . Anything between the start and end tags is ignored by the browser and not displayed on the page. You can use this to leave notes for yourself or others who might read your code later.

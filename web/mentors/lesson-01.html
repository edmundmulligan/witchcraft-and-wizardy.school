Lesson 1 - Creating Your First Web Page

Introduction

This lesson introduces students to the basics of web development by guiding them through the process of creating their first web page. It is designed to be engaging and hands-on, allowing students to see immediate results from their efforts.

Learning Outcomes

By the end of this lesson, students will have:

  • Started using Visual Studio Code as their code editor.
  • Created a simple web page using HTML.
  • Learned how to structure content with headings and paragraphs.
  • Learned about comments in HTML.

Resources

Here are some resources that can help you as you guide your students through this lesson:

  • YouTube - Visual Studio Code - A tutorial video for getting started with Visual Studio Code.
  • MDN Web Docs - HTML - A comprehensive resource for learning about HTML.
  • W3Schools - HTML - A beginner-friendly resource for learning HTML.
  • W3Schools - HTML Headings - A guide to using headings in HTML.
  • W3Schools - HTML Paragraphs - A guide to using paragraphs in HTML.
  • W3Schools - HTML Comments - A guide to using comments in HTML.
  • Glossary - A glossary of web development terms used in the lessons.

Challenges

Encourage students to experiment with different HTML elements as they create their web page. Remind them that the goal is to have fun and learn, so they should feel free to be creative and make the page their own.

If they need inspiration for what to type in their paragraphs, they can look at the interesting facts about witches and wizards in the facts page.

Lesson 4 - Images

Introduction

In this lesson, students are introduced to images in HTML. Images are used to enhance the visual appeal of a web page and to convey information more effectively. They help to create a more engaging and informative experience for users.

The most likely problem students will encounter is saving images into the correct folder and using the correct file path in their HTML code. Encourage them to check that they have saved their image files in the same folder as their HTML file, and that they are using the correct file name and extension in their code.

Learning Outcomes

By the end of this lesson, students will have:

  • Understood the concept of images in HTML.
  • Identified different types of images and their purposes.
  • Added images to a web page to enhance its visual appeal and convey information.

Resources

Here are some resources that can help you as you guide your students through this lesson:

  • W3Schools HTML Images
  • MDN Web Docs - Image Types
  • Adobe Guide to Image File Formats

Challenges

Encourage students to experiment with different types of images as they create their web page. Remind them that the goal is to have fun and learn, so they should feel free to be creative and make the page their own.

Encourage them to create a page that tells a story using words and images together.

By now they will probably be familiar with the text and image resources on this website so encourage them to search the web for images that they can use. They could also take their own photographs and use them, although they should avoid taking photos of people and using them without getting permission first.

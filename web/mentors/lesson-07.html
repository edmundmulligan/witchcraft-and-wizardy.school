Lesson 7 - Firefox Developer Tools

Introduction

In this lesson, students use Firefox Developer Tools to inspect the HTML they have already written in lessons 1-6. The focus is on the Inspector tab only. Students are not expected to use CSS tools or the JavaScript Console yet.

Learning Outcomes

By the end of this lesson, students will have:

  • Opened Firefox Developer Tools and selected the Inspector tab.
  • Used element picking to map visible page content to HTML tags.
  • Read and interpreted key HTML attributes such as src and alt.
  • Diagnosed a missing image by checking for filename and path mistakes.

Resources

Here are some resources that can help you as you guide your students through this lesson:

  • Firefox DevTools - Page Inspector
  • MDN - What are browser developer tools?
  • MDN Web Docs - HTML <img> Element

Challenges

Encourage students to inspect their own pages from lessons 1-6 and describe what they see in the HTML tree. Ask them to explain how the visible content maps to elements such as <h1>, <p>, <section>, and <img>.

A useful challenge is to introduce a small typo in an image filename, then have students use Inspector to find the incorrect src value and repair it. This builds confidence in practical debugging without requiring CSS or JavaScript knowledge.

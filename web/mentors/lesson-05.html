Lesson 5 - Hyperlinks

Introduction

In this lesson, students are introduced to hyperlinks in HTML. Hyperlinks are used to navigate between different web pages or sections within a page. They are a fundamental part of web development and help create a connected web experience.

Learning Outcomes

By the end of this lesson, students will have:

  • Understood the concept of hyperlinks in HTML.
  • Identified different types of hyperlinks and their purposes.
  • Created a simple web page using hyperlinks.

Resources

Here are some resources that can help you as you guide your students through this lesson:

  • W3Schools HTML Links
  • MDN Web Docs - HTML <a> Element

Challenges

Encourage students to experiment with different HTML links as they create their web page. Remind them that the goal is to have fun and learn, so they should feel free to be creative and make the page their own.

If they have created multiple pages, they can try linking them together using hyperlinks.

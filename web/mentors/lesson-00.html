Lesson 0 - Getting Started

Student Lesson 00

Introduction

This is probably the longest and most intricate lesson of the whole course, but it is very important to enable the other lessons. Both you and the student may find it challenging and you will probably spend a significant amount of time problem solving and troubleshooting. Depending on the student's ability, you may want to install the required software yourself and have them skip lesson 0 completely. There really aren't any learning outcomes for this one so you won't impact their learning much if you do it for them.

Not all the tools need to be installed at the beginning. As a minimum, the student will need a code editor and a web browser. The other tools (Git, Node.js, and SQLite) are only needed for later lessons so they can be installed later if preferred. You are welcome to encourage the student to use your preferred tools (such as Atom code editor or Google Chrome) if you will find it easier to support them, but be aware the other lessons will assume the tools listed in the instructions, and you may be creating extra work for yourself.

Warning. This lesson involves installing software on the student's computer. If they are using a shared or school computer, please check with a parent, guardian, or teacher before proceeding with the installations. (Of course, that may be you, in which case the problem is fairly academic). Some of the installation steps require administrative privileges and you may want to do those steps yourself to avoid giving the student the super user password. Some computers (particularly those that are managed by a school or corporate IT department) are locked to prevent unapproved software installations. If that is the case, you will not be able to complete this lesson on that computer and the student will not be able to do any of the exercises. They can still read the lessons and learn the theory, but they will not be able to practice the skills.

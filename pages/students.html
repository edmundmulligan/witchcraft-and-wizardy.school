Lessons

Lesson 0: Setting up

Set up your computer with the tools required to do the lessons. If you are using a club computer at a Code Dojo then this will have alredy been done and you may skip this lesson.

Lesson 1: Your first web page

Build a simple web page using HTML.

Lesson 2: Landmarks

Add landmarks and other structural elements to your web page.

Lesson 3: Lists

Add lists to your web page, both ordered and unordered.

Lesson 4: Images

Add some images to your web page.

Lesson 5: Tables and hyperlinks

Add tables and hyperlinks to your web page.

Lesson 6: Introducing CSS

Learn about Cascading Style Sheets (CSS) and how they can change the look of your web pages.

Lesson 7: External CSS

Learn how to use an external CSS stylesheet to change the look of your web pages and separate content from presentation.

Lesson 8: CSS and lists

Make your lists more visually appealing using CSS.

Lesson 9: Links and CSS

Make your hyperlinks more visually appealing using CSS.

Lesson 10: Responsive Design

Learn about media queries and how they can be used to create responsive web pages that work on both desktop and mobile devices.

Lesson 11: Javascript

Use JavaScript to add interactivity to your web pages.

Lesson 12: External Javascript files

Just as CSS can be put in external files, so can JavaScript. In this lesson you will learn how to use external JavaScript files.

Lesson 13: Errors and Exceptions

In this lesson you will learn about errors and exceptions in JavaScript and how to handle them.

Lesson 14: Testing and Debugging

Learn how to test and debug your JavaScript code using the browser's developer tools.

Lesson 15: Forms

Learn how to create forms to collect information from users.

Lesson 16: Cookies

In this lesson you will learn about cookies and how they can be used to store information on the user's computer.

Lesson 17: Clients and Servers

In this lesson you will learn about the client- server model of the web.

Lesson 18: A server page

In this lesson you will learn how to create a simple server page using a server-side language.

Lesson 19: Processing Forms on the server

Learn how to process the data you collect from forms on the server.

Lesson 20: Databases

In this lesson you will learn about databases and how to use them to store data for your web applications.

Lesson 21: Security

In this lesson you will learn about basic web security concepts,

Lesson 22: Passwords

Passwords require special handling to keep them secure. In this lesson you will learn how to handle passwords safely.
